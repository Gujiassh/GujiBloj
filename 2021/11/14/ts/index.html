TS笔记

Author: GUJI Date: November 14, 2021  9:53:17 Category: js

基础类型

Void

代表没有任何返回值

 ## Object

定义一个函数，参数是obj，返回值也是object

如果想传入一个其他数据类型可以这样：

getObj(new String(123))

getObj(String) // 也是可行的

联合类型

取值可以是多种类型中的一种

类型断言

不进行特殊的数据检查和结构

方法一： <类型>

方法二： as

类型推断

自动推断类型

定义了类型没有赋值时，该变量是any类型

接口

接口用来对值进行类型检查， 使用接口来定义对象的数据类型（约束）

函数类型

使用接口 定义函数的类型

类类型

继承

使用extends继承

子类构造函数中使用super调用父类的构造方法，子类可以调用或重写父类的方法

多态

实例化子类实例时使用父类型

public private protected readonly

描述类中的成员属性的可访问性

readonly 只读属性不能在外部修改

存储器 getter setter

静态属性

在类中使用 static 修饰

使用类名. 的方式获取值

抽象类

没有具体实现

不能被实例化

关键字： abstract

函数

标明参数类型，返回值类型

可选参数和默认参数，剩余参数

可选参数： 加问号 —-> name?:string

默认参数: 加初始值 –> name:string=’王’

剩余参数：剩余参数放在了一个字符串数组里面,放在所有参数最后 –> …args:string[]

函数重写

这样，两个不同的参数就用相同方法名实现了

泛型

定义接口，函数，类的时候不能确定传入数据的类型，使用的时候才能确定的类型

不确定时，如果使用any， 那么会导致丢失了传入数据的信息，例如：数据的类型

使用泛型：

多个泛型参数的函数

声明文件

引入第三方库使用

declare var jQuery : (selector:string) => any
